Anchor v1 is the stable, published Anchor framework for building Solana programs and clients. It simplifies the process of writing, testing, deploying, and interacting with Solana programs.

The Anchor framework helps developers build production-ready applications faster while reducing potential vulnerabilities through built-in security features.

<Callout variant="note" title="Anchor v2 alpha">
  Anchor v2 (`anchor-lang-v2`) is in alpha on the [`anchor-next`](https://github.com/solana-foundation/anchor/tree/anchor-next) branch. v2 is built on pinocchio and `#![no_std]{:rs}`, producing up to 94% smaller binaries and 3 to 6 times fewer CU per instruction. The v1 path documented here remains the stable, published path. See the [v2 docs](/docs/v2/) for the alpha reference.
</Callout>
