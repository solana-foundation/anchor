title: SPL token basics
description: Learn how to integrate SPL tokens into your Solana programs using the Anchor framework.
---

This section covers the basics of interacting with SPL tokens in Anchor programs, focusing on the most commonly used instructions. All examples work identically with both the original Token Program and the Token Extension Program (Token 2022), which share the same base implementation.

- [Create a mint account](/docs/tokens/basics/create-mint) initializes a mint that defines a new token.
- [Create a token account](/docs/tokens/basics/create-token-account) initializes an account that holds a balance of a specific mint.
- [Mint tokens](/docs/tokens/basics/mint-tokens) increases the supply of a mint and credits a token account.
- [Transfer tokens](/docs/tokens/basics/transfer-tokens) moves tokens between two token accounts of the same mint.
